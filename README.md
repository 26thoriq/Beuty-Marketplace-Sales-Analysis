
# Beauty Marketplace Analytics

## 📝 Project Overview

This repository contains an end-to-end data analytics project for a Beauty Marketplace. The objective of this project is to analyze multi-channel sales performance, product profitability, customer behavior, and operational efficiency to support strategic business growth.

## 🛠 Project Structure

/Datasets
Contains the raw data files used for the analysis.

/Image
Contains visual assets and dashboard snapshots for documentation and presentation.

/Power BI
Contains the interactive Power BI (.pbix) dashboard file.

/SQL script
Contains the SQL queries used for data extraction, cleaning, transformation, and analysis.


## 🚀 Key Features

### Sales Performance

Multi-channel sales analysis covering marketplace, website, and offline store performance, with a focus on revenue contribution and profit margin optimization.

### Product Performance

SKU-level analysis, product profitability tracking, best-selling product identification, Pareto analysis, and skin-type-based market insights.

### Customer Intelligence

Customer segmentation, demographic profiling, purchasing behavior analysis, and customer satisfaction tracking.

### Operational Insights

Supply chain performance analysis, return reason breakdown, delivery performance monitoring, and regional warehouse performance audit.

## 📊 Technical Stack
* Google sheet (Data understanding)

*  Database: SQL Server(data modeling,data cleaning and exploratory data analysis)

* Visualization: Power BI(interactive dashboard,dax).
